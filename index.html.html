الكيوبت (Qubit) — وحدة الحوسبة الكمية

البت العادي بحاسوبك إما 0 أو 1. الكيوبت يكون 0 و1 في نفس الوقت — هذي ظاهرة التراكب الكمي (Superposition). وعند ربط كيوبتين سوية بـالتشابك الكمي (Entanglement)، قياس أحدهما يحدد الثاني فوراً.
